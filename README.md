A full-stack web application for managing machines, maintenace tasks, and monitoring system data.
Built using Node.js, Express, typeScript, Prisma, and a frontend client application.

Features
Machine management (add, view, update, delete)
Rest Api backend using express 
Database integration using Prisma ORM
End to-end testing with Cypress
Unit/function tests for backend logic .

Tech stack
Backend
Node.js
express.js
Typescript
Prisma ORM

Frontend
client application (javaScript/TypeScript)
public + Src structure

Testing
Cypress (end-to-end testinh)
Unit tests

Database
PostgreSQL/SQLite (base on setup)
